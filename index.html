Proyectos

Blog /Ask.Ans.Run

Un sencillo foro de texto e imagenes usado para resolver dudas sobre codigo.
Viene incorporado con un sistema de registro e inicio de sesion asi como actualizaciones de post en tiempo real

Sistema de compras app Paris.com

Aplicación web diseñada para optimizar la experiencia de usuario en el flujo de compra.
Incluye un carrito de compras dinámico, integración de pasarelas de pago y un panel de seguimiento de pedidos que actualiza el estado de la transacción de forma inmediata.

Control interno de stock supermercados Jumbo

Plataforma de gestión de inventarios desarrollada para mejorar la logística interna.
Permite el monitoreo de entrada y salida de productos en tiempo real, generación de alertas automáticas para reposición y reportes detallados para la toma de decisiones estratégicas.
