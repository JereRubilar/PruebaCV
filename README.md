# **CV Interactivo fundamentos front-end :page_facing_up:**

Este CV ha sido creado con herramientas fundamentales de desarrollo web front-end  

En él podemos encontrar diferentes ejemplos de diseños modernos, responsivos y dinamicos que apuntan a que la tarea de analizar el CV no sea aburrida y tediosa, si no que invite al lector a interactuar y ser parte del proceso al momento de descubrir las habilidades o aptitudes del postulante.

Es importante recalcar que lo contenido en este CV no corresponde a la verdad y solo es usado de ejemplo para el ejercicio.
